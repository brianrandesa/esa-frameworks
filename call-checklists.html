The 7 Rules — hold these on every call

  1. Lead with truth — never guarantee.
  2. Check real engagement (story views, likes, comments) — not follower count.
  3. Same numbers every time: 165-point checklist · $30–50 / registrant · 30% show · 30% book · 30% close.
  4. Posture — "I don't need it that bad that I'd take it without knowing we can help you."
  5. FTC-safe language, always — illustrative ranges, never "you'll make $X."
  6. Never drop the price — no downsell.
  7. Ask for the sale, then go silent — and never hang up without booking the next step.
ROI flip — fill this in before the call:
Their client is worth $[____]/yr  →  just 2 clients = $[____]  →  "$15K is a rounding error against what one webinar can return."
Keep it FTC-safe: "conservative industry ranges," "here's the math, not a promise."

path A · they're in The Money Sequence

Verbal yes / ready for paper. Run it same day — a yes decays by the hour.

path B · "let me think" The Follow-Up Sequence

"I need to think about it" is not a no — it's an unfinished yes. Most deals are won or lost right here. Never leave it to "I'll reach out."

⚡ Automate It — two GHL Workflows

Don't run the busywork by hand. Build one Workflow per path, each triggered by the stage change — so you only touch the human + money steps.

Workflow 1 — trigger: stage → "Verbal Yes" → auto-send the SMS, wait 5 min, send the recap email, create a "send proposal + contract + invoice" Task, notify you + Viktor.
Workflow 2 — trigger: stage → "Follow-Up" → run the Day 1–14 drip above; it stops the moment they book or move to Won / Lost.
You keep manual: review & send the contract, collect the payment, and book the next call live on the phone. Fire the proposal / contract / invoice from the deal yourself so you control the moment.
